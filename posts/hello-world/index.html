Starting a new journey.

November 04, 2020

This is the first step into learning the technologies I want to learn. The first step into a better self organization and discipline. If this blog lives, then my journey is going through the right way, if this is dead, then that means I failed.

Both technologies are quite awesome, though Rust requires a lot of effort to get it right, and probably because of this my existing backend code is not as good as it could be. Hopefully, this time I can change that.
